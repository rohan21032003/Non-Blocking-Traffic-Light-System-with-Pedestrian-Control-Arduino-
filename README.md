# Non-Blocking-Traffic-Light-System-with-Pedestrian-Control-Arduino-

This project implements a traffic light control system using Arduino with non-blocking timing logic based on millis().

Unlike traditional delay-based implementations, this system uses a state machine approach to manage traffic flow and pedestrian requests efficiently without blocking execution.

Features:
- Non-blocking timing using millis()
- State machine-based traffic control (RED, YELLOW, GREEN)
- Pedestrian request system using push button
- Immediate override for pedestrian crossing
- Clean and modular embedded logic design

Key Concepts:
- Embedded C/C++ programming
- Real-time system design
- Event-driven programming
- State machine implementation

Future Improvements:
- Multi-direction (2-way / 4-way) traffic control
- Queued pedestrian request handling
- Sensor-based traffic optimization
