# Non-Blocking-Traffic-Light-System-with-Pedestrian-Control-Arduino-
Arduino-based traffic light controller using millis() for non-blocking timing. Implements a finite state machine with pedestrian override via push button. Focuses on real-time responsiveness and embedded system design principles.
